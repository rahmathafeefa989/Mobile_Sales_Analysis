# Mobile Sales Dashboard – Power BI

## 📊 Project Overview
This project is an interactive **Mobile Sales Dashboard** built using **Microsoft Power BI**. The dashboard provides insights into mobile sales performance, customer ratings, transaction methods, and brand performance.

The goal of this project is to practice **data analysis, visualization, and dashboard development** while building practical skills required for a **Data Analyst role**.

## 🚀 Features
- Total Sales, Quantity, Transactions, and Average Sales KPIs
- Sales analysis by day
- Brand-wise sales performance
- Mobile model sales comparison
- Payment method distribution
- Customer rating analysis
- Monthly interactive filters

## 📈 Dashboard Insights
The dashboard helps to:
- Track overall mobile sales performance
- Identify top-performing brands and models
- Analyze customer purchasing behavior
- Understand payment method preferences
- Evaluate customer satisfaction through ratings

## 🛠 Tools Used
- **Power BI**
- Data Visualization
- Data Analysis
- Interactive Dashboard Design
